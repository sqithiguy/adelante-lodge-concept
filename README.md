# Adelante Lodge Redesign Concept

Standalone multi-page concept for a premium, mobile-first redesign of Adelante Lodge in Bloemfontein.

## Files

- `index.html` homepage
- `rooms.html` room types
- `conference.html` conference facility
- `explore.html` Bloemfontein highlights
- `gallery.html` gallery concept
- `contact.html` contact and booking page
- `styles.css` shared styling
- `script.js` mobile menu behavior

## Notes

- Uses the supplied logo and exterior image from the local uploads.
- Keeps the existing business facts, booking route and contact details while rewriting the copy.
- Built as handover-friendly static HTML/CSS/JS.
